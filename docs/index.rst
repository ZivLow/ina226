Welcome to Ziv Low's INA226 documentation!
==========================================

This library component provides an easier way to use the INA226 power monitor IC.
This C++ library is written specifically for the ESP-IDF platform (at least ESP-IDF v5.2).

Take a look at the provided example code for inspiration on how to use this library.
